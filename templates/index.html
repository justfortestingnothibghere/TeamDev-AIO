API Endpoints

Everything You Need

Clean, consistent REST endpoints. Pass your API key and a URL — get back rich structured data.

GET
/api/v1/dl
AIO Universal Downloader
Supports YouTube, TikTok, Instagram, Facebook, Twitter/X, Vimeo, Dailymotion, Pinterest, Bandcamp, and 35+ more sites via yt-dlp.
?url= ?api= title medias[] thumbnail
GET
/api/v1/s
Spotify Track Downloader
Fetch full MP3 downloads for any Spotify track URL. Returns artist, title, album art, and a direct download link.
?url= ?api= mp3 artist thumbnail
GET
/api/v1/xham
XHamster Extractor
Multi-quality m3u8 stream extractor for XHamster. Returns adaptive streams at multiple resolutions.
?url= ?api= streams{} m3u8
GET
/api/v1/xham/search
XHamster Search
Search XHamster by keyword and receive a structured list of results with titles, thumbnails, and video links.
?q= ?api= results[]
GET
/api/v1/hcity
HentaiCity Stream Extractor
Extracts m3u8 stream URLs from HentaiCity. Returns title, thumbnail, trailer, and the main stream link.
?url= ?api= m3u8 trailer
GET
/api/v1/phub
PornHub Downloader
Download from PornHub via yt-dlp using .org mirror routing. Returns direct MP4 links at multiple quality levels.
?url= ?api= formats[] duration
Usage Examples

Get Started in Seconds

Copy and paste. Works with any HTTP client, any language.

# AIO Universal Downloader curl "https://aio-production-df67.up.railway.app/api/v1/dl ?url=https://youtube.com/watch?v=dQw4w9WgXcQ &api=td_YOUR_KEY_HERE" # Spotify Download curl "https://aio-production-df67.up.railway.app/api/v1/s ?url=https://open.spotify.com/track/TRACK_ID &api=td_YOUR_KEY_HERE"
import requests BASE = "https://aio-production-df67.up.railway.app/api/v1" API_KEY = "td_YOUR_KEY_HERE" params = { "url": "https://youtube.com/watch?v=dQw4w9WgXcQ", "api": API_KEY } res = requests.get(f"{BASE}/dl", params=params) data = res.json() print(data["title"]) print(data["medias"][0]["url"])
const BASE = 'https://aio-production-df67.up.railway.app/api/v1' const API_KEY = 'td_YOUR_KEY_HERE' async function download(url) { const params = new URLSearchParams({ url: url, api: API_KEY }) const res = await fetch(`${BASE}/dl?${params}`) const data = await res.json() console.log(data.title) console.log(data.medias[0].url) }
$base = 'https://aio-production-df67.up.railway.app/api/v1'; $key = 'td_YOUR_KEY_HERE'; $url = 'https://youtube.com/watch?v=dQw4w9WgXcQ'; $endpoint = $base . '/dl?url=' . urlencode($url) . '&api=' . $key; $res = file_get_contents($endpoint); $data = json_decode($res, true); echo $data['title'] . "\n"; echo $data['medias'][0]['url'];

FAQ

Common Questions

Everything you need to know before getting started.

How do I get an API key? ⌄
Contact us on Telegram at @MR_ARMAN_08. We'll generate a key for you and configure your rate limits based on your use case.
What is the rate limit? ⌄
Default is 60 requests per 60 seconds per API key. Custom limits can be configured by admin. Contact us for higher limits.
Can I use it without an API key? ⌄
This depends on whether API enforcement is enabled. When enforcement is off, the API key is optional and all requests are treated as anonymous. When enforcement is on, a valid key is required.
What platforms does /api/v1/dl support? ⌄
It supports 35+ platforms via yt-dlp including YouTube, TikTok, Instagram, Facebook, Twitter/X, Vimeo, Dailymotion, Pinterest, Bandcamp, Reddit, Twitch, SoundCloud, and many more.
What does the response include? ⌄
Responses include success status, source platform, title, author, thumbnail URL, duration in seconds, media type, and a medias array with quality, extension, direct URL, and file size for each available format.
Is there a self-hosted option? ⌄
Yes. The project is built with FastAPI and can be deployed on any server or platform that supports Docker or Python. Check the Dockerfile in the repository for deployment guidance.
